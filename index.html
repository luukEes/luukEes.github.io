About

Motivated engineer with a Master's degree in Mechanics and Machine Construction, bringing nearly 5 years of experience in maintenance automated production line and programming industrial robots. Involved in collaborative problem-solving with the automation team, enhancing production efficiency.

In 2023, I started developing towards object-oriented programming, choosing JAVA. So I completed a programming course, after which I learned the following topics: Java, OOP, SQL, Servlets, JSP, JavaScript, Hibernate and Rest API. Every day I expand my knowledge related to programming in Java.

At the moment I started to create simple applications in MVC architecture using Spring. As part of my free time, I implement new functionalities and deal with the topic of exception handling.

Before that I got familiar with JPA and Hibernate libraries. I wrote a simple application to connect to MySql database, which performs saving of objects on the server. In addition to the backend layer, I am currently learning other technologies, i.e. Java Script, HTML, CSS. I gained the basics of all issues related to programming in Java on the Coders Lab - Java Developer course.

In my case, programming with the passage of time while learning, became not only a goal to achieve, but mainly my hobby, which I regularly practice in every free moment. The biggest goal at the moment is to gain the first commercial experience
